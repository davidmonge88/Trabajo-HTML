El código HTML es un lenguaje diseñado para estructurar semánticamente documentos para web. Basa su funcionamiento en el empleo de etiquetas de acotación <...> para abrir un elemento y para cerrarlo. similar a la acotación física que realizar los Marbetes Sí entre un grupo de 2 etiquetas no consta información, hablamos de un elemento vacío. Cada elemento puede contener atributos propios para darle distintas propiedades. Herramientas •Navegador •Editor •Analizador sintáctico •Procesador Formato •Declaración de documento •Inicio •Cabecera •Cuerpo Fin vas a tener que poner < para indicar < y > para indicar >
